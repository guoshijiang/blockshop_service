### 2.项目开发部署

第一步：克隆代码

```bigquery
git clone git@github.com:guoshijiang/blockshop_service.git
cd blockshop_service
```

第二步：数据库 migrate

去 models 下面的 base.go 里面打开 `RunSyncdb` 函数，自动生成数据库，注意配置 false 和 true 项

第三步：运行开发
```bigquery
bee run 即可运行代码进行开发
```

第三步：代码部署

```bigquery
go build 完成之后，使用进程管理管理启动服务即可
```
